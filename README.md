# 🧭 后台权限管理系统

一个基于 **Spring Boot + Redis + MySQL** 的快速开发后台权限管理系统，提供用户管理、角色管理、菜单权限、登录认证等基础功能，帮助开发者快速搭建管理后台。

---

## 🌟 功能特性

- 👤 用户管理：支持用户的创建、修改、删除、禁用等操作
- 🧩 角色管理：灵活分配角色与权限
- 🔐 权限控制：基于角色的动态权限系统（RBAC）
- ⚙️ 菜单管理：动态路由加载，支持前后端分离架构
- 📊 日志审计：记录用户操作与登录日志
- 💾 Redis 缓存：提升系统性能与会话管理效率
- 🧱 统一接口响应：标准化接口返回格式，便于前端对接
